With the user profile forms implemented, tailoring a CV for the user is now possible. The CV creation/fine tuning feature will pull in the user data and create a PDF using the api from https://latex.ytotech.com, using a LaTeX template taken from my own personal [cv](https://www.davidculemann.com/cv).
